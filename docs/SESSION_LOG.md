# Session Log

Brief notes from each working session. Keep entries short — a few bullet points is enough.

The goal is to be able to resume quickly after a break and to give Claude
useful context at the start of the next session.

---

## Format

### YYYY-MM-DD

**Worked on:** _[Feature or task name]_

**Completed:**
- _[Thing 1 finished]_
- _[Thing 2 finished]_

**Blockers / open questions:**
- _[Anything unresolved or that slowed progress]_

**Next session:**
- _[The single most important thing to pick up next]_
